Library Management System – Srushti Technologies

About Project:

The "Library Management System" is a web-based application designed to manage and automate library operations in an efficient way.
It helps in handling books, students, issue/return records, and search functionality, reducing manual work and improving accuracy.

Features:

* Add new books to library
* Manage student/user records
* Issue books to students
* Return book management
* Search books easily
* Track available and issued books
* Simple and user-friendly interface

Tech Stack:

* JavaScript

Backend
* Python (Flask)
*

Project Objectives:

* Automate library management system
* Reduce manual record keeping
* Improve book tracking system
